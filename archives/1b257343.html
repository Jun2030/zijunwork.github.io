Mysql-常用语句

发表于 2020-09-10 分类于 服务端 ， Database ， Mysql 阅读次数： Valine：
本文字数： 2.5k 阅读时长 ≈ 5 分钟

本篇主要记录Mysql-常用语句。

以下语句，注意要携带语句后的分号

连接数据库报错：UnhandledPromiseRejectionWarning: Error: ER_NOT_SUPPORTED_AUTH_MODE: Client does not support，mysql8.0加密方式原因。

  1. win键搜索，打开 MySQL 8.0 Command Line Client
  2. 键入密码：123456
  3. 修改加密规则：ALTER USER 'root'@'localhost' IDENTIFIED BY 'password' PASSWORD EXPIRE NEVER;
  4. 更新用户密码：ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';
  5. 刷新权限：FLUSH PRIVILEGES;

操作数据库

  • mysql -u root -p 发起连接数据库，如果是用win键搜索，【MySQL 8.0 Command Line Client】打开，则无需发起，直接可输入密码
  • SHOW DATABASES; 显示所有数据库
  • SHOW CREATE DATABASE test; 显示指定（名为test）数据库
  • CREATE DATABASE db; 创建（名为db）的数据库（简单版），之后为了避免乱码，需要改变数据库编码方式，推荐使用严谨版
  • ALTER DATABASE db DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci; 改变数据库编码方式，一般用utf8，避免乱码
  • CREATE DATABASE IF NOT EXISTS db DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci; 有条件地创建（名为db）数据库（严谨版）
  • DROP DATABASE db; 删除（名为db）数据库，一次只能删除一个数据库
  • USE db; 使用（名为db）数据库

操作表

  • SHOW TABLES; 显示所有的表，前提是 USE db;，使用了当前的数据库
  • CREATE TABLE IF NOT EXISTS user(id INT, name VARCHAR(20), age INT); 创建表（名为user），变量在前，数据类型在后，变量与变量之间用逗号隔开
  • CREATE TABLE user SELECT * FROM users; 根据现有的表（名为users）来创建表（名为user）并插入指定条件的数据
  • DESC user; 查看表（名为user）的结构
  • SELECT * FROM user; 查询指定的表（名为user），* 代表所有
  • SELECT id FROM user; 查看user表中的id列
  • ALTER TABLE user ADD phome INT; 在表（名为user）中，增加一个列（phone），类型为int
  • ALTER TABLE user DROP phone; 在表（名为user）中，删除一个列（phone）
  • RENAME TABLE user TO users; 表名重命名（user -> users）
  • ALTER TABLE user CHANGE phone phonenum INT; 修改表（名为user）中的列名（phone -> phonenum）
  • DROP TABLE IF EXISTS user; 删除指定表（名为user）

操作表中数据

  • INSERT INTO user(id, name, age) VALUES (2, "me", 10); 向表（名为user）插入数据，前一个括号是列名，后一个括号是对列名的赋值
  • INSERT INTO user VALUES (1, "DEVIN", 20); 如果是对表（名为user）中所有列赋值，可以省略前一个括号，直接values
  • UPDATE user SET id=3 WHERE id=1; 修改表（名为user）中 id=1的那一行数据，变更为 id=3
  • DELETE FROM user WHERE id=3; 删除表（名为user）中 id=3的那一行数据

查询表中数据

条件查询

  • SELECT * FROM user WHERE id=2; 查询表（名为user），条件为id=2
  • SELECT * FROM user WHERE id BETWEEN 0 AND 10; 查询表（名为user），条件为 0<=id<=10

模糊查询

% 代表多个任意字符，_ 代表任意一个字符

  • SELECT * FROM user WHERE name like "%un%"; 查询表（名为user），条件为 name 字段中含有“un”的一条数据

分页查询

  • SELECT * FROM user LIMIT 0,1; 查询表（名为user），第0条到第1条数据，即第一条数据

去重查询

  • SELECT DISTINCT age from user; 查询表（名为user），字段age的去重查询

连接查询

  • SELECT * FROM user INNER JOIN student ON user.id=student.id; 内连接，查询两个表（user,student）的id交集部分数据

排序数据

升序：ASC，降序：DESC，默认为 ASC

  • SELECT * FROM user ORDER BY age,name ASC; 查询表（user）的所有数据，排序为age的升序，如果age相同，则按name的升序

聚合函数

  • SELECT COUNT(id) FROM user; 查询表（user）中有多少条数据
  • SELECT MAX(age) FROM user; 取表（user）中，age的最大值
  • SELECT MIN(age) FROM user; 取表（user）中，age的最小值
  • SELECT SUM(age) FROM user; 取表（user）中，age的总和
  • SELECT AVG(age) FROM user; 取表（user）中，age的平均值

操作键

  • ALTER TABLE user ADD RPIMARY KEY (id) 添加主键（primary key，主键只有一个）
  • ALTER TABLE user DROP RPIMARY KEY 删除主键
