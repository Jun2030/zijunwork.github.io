全局属性

ctx 作为上下文对象，可以用于共享参数，官方提供了2个全局属性：

  • context
  • state

context

context 是作用在 app 这个对象上属性，即Koa的一个实例，它类似js中对象的 prototype 属性，在 context 上添加的属性，可以通过 ctx 对象调用。

  • context 上一般添加一些全局的常用方法或属性，如封装的 axios 请求等。

state

state 是作用在 ctx 对象上的，也就是 ctx 的属性。其不同于 context,state 是一个空对象，不用担心属性覆盖问题。常用与特性检测场景：

  • 语言检测，检测客户端能支持什么样的语言，可以用来做全球化多语言适配

  • 数据类型检测，检测浏览器支持的最佳网页类型

  • 编码检测，检测浏览器支持什么编码

  • 字符集检测，用于检测浏览器支持什么样的字符格式
