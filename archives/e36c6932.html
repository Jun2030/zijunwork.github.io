配置编辑器

  1. 配置ESLint：选择 设置 / 语言和框架 / JavaScript / 代码质量工具 / ESLint
    1. 勾选 [手动 ESLint 配置]
    2. 勾选 [保存时运行 eslint –fix(U)]
  2. 配置ESLint：选择 设置 / 语言和框架 / JavaScript / Prettier
    1. 选择 [Prettier]包
    2. 增加文件运行 {**/*,*}.{js,ts,jsx,tsx,vue}
    3. 勾选 [代码重新格式化时]
    4. 勾选 [保存时]
  3. 配置TSLint: 选择 选择 设置 / 语言和框架 / TypeScript / TSLint
    1. 选择TSLint包
    2. 注意不要勾选 [Lint JavaScript文件]
  4. 鼠标右键 [src]目录，点击[修复ESLint问题]，即可生效（如果未生效，则需要重启Webstorm）
  5. 测试运行 yarn serve 、 yarn build，无问题，此时，提交一个Git版本
